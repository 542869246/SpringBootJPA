# SpringBootJPA
springbootjpa

# 说明
使用spring boot+jpa+thymeleaf+mysql做的增删改查小玩具

## 技术点
- spring data jpa
- thymeleaf模板
- springboot的定时任务
- 邮箱配置
- swagger
- Jenkins部署
- Spring Boot Actuator监控应用
- 整合Druid
- 上传下载


## 注意事项
- jdk1.8
- spring boot 1.4.1
- 127.0.0.1:8080/demo/findAll进入
- 127.0.0.1:8088/monitor进入Actuator  账号：admin 密码：123456
- 127.0.0.1:8080/druid进入Druid页面   账号：druid 密码：123456
- 数据库mysql
