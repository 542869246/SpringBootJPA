# SpringBootJPA
springbootjpa

# 说明 #
使用spring boot+jpa+thymeleaf+mysql做的增删改查小玩具

## 技术点 ##
- spring data jpa
- thymeleaf模板
- springboot的定时任务
- 邮箱配置
- swagger
- Jenkins部署
- Spring Boot Actuator监控应用
- 上传下载


## 注意事项 ##
- jdk1.8
- spring boot 1.4.1
- 127.0.0.1:8080/进入
- 因为设置了Actuator 访问需要账号密码  账号：admin 密码：123456
